Featured AIML Projects

This section highlights projects that demonstrate my capability in end-to-end AI/ML model development, from data pre-processing to model deployment and evaluation.

1. Advanced Image Classification using CNN

This project involved designing and training a custom **Convolutional Neural Network (CNN)** architecture to classify images across hundreds of categories. I used **Transfer Learning** with a pre-trained ResNet model to achieve rapid convergence and optimized the training pipeline using Keras/TensorFlow. The final model achieved a **Top-5 accuracy of 97.2%**, demonstrating strong generalization capabilities and robust handling of image variance. This showcases my deep understanding of deep learning topologies and performance tuning.

2. Conversational Chatbot using NLP

I engineered a fully functional, intent-based **chatbot** designed to handle customer queries for a simulated e-commerce platform. The solution utilizes **Natural Language Processing (NLP)** techniques, including tokenization, stemming, and TF-IDF vectorization, combined with a neural network classifier to accurately determine user intent. The system provides pre-scripted responses and dynamic data retrieval, highlighting my skills in text data processing and practical deployment of conversational AI.

3. Financial Predictive Analytics (Customer Churn)

The objective of this project was to predict customer churn in a telecommunications dataset. I performed extensive **Exploratory Data Analysis (EDA)**, handled class imbalance using SMOTE, and benchmarked various traditional **Machine Learning algorithms** (Logistic Regression, Random Forest, XGBoost). The final **XGBoost** model provided the highest performance, achieving an **AUC-ROC score of 0.88**. The key outcome was identifying the most significant features influencing churn, providing clear insights for targeted marketing strategies.

4. Real-time Recommendation System

I built a prototype for a **Collaborative Filtering Recommendation System** using the Surprise library in Python. The system processes user-item interaction data to generate personalized recommendations. I implemented both **User-Based** and **Item-Based** collaborative filtering and evaluated performance using Root Mean Square Error (RMSE). This project demonstrates expertise in building data pipelines for large-scale personalization and understanding memory-based recommendation models.
